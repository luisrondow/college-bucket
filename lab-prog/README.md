# Programming Lab

Hands-on programming exercises and projects in C.

## Activities

- [AF0_2](./af0_2.c) — Reads a word from a text file and checks whether it is a palindrome (3–10 characters)
- [AF1_1](./af1_1.c) — Detects palindromes in sentences, ignoring spaces, punctuation, and accents, and outputs the original formatting
- [AF1_2](./af1_2/) — Manages student records with dynamic memory allocation: add students, list all, and calculate grade averages
- [AF2_1](./af2_1/) — Header file exercise: calculator module with multiply and subtract declarations
- [AF2_2](./af2_2.md) — Identifies and fixes a missing header guard (`#ifndef`/`#define`) in a `.h` file
- [AF2_3](./af2_3/) — Geometry library with area functions for circles and rectangles, compiled and tested separately
- [AF2_4](./af2_4/) — Refactors a monolithic program into modular files: stats module with add/average functions
- [AF2_5](./af2_5.md) — Code review exercise: identifies and fixes multiple issues in a C module (header guards, extern, static linkage)
- [AF2_6](./af2_6/) — Multi-file project with a logging library (log messages, errors, toggle) using `include/`+`src/` structure
